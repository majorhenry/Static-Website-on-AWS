Deploy Static Website on AWS
Website URL:
S3 URL: http://travelblogv1.s3-website-us-east-1.amazonaws.com

In this project, deploy a static website to AWS using S3, CloudFront, and IAM.

The files included are: 

index.html - The Index document for the website.
/img - The background image file for the website.
/vendor - Bootssrap CSS framework, Font, and JavaScript libraries needed for the website to function.
/css - CSS files for the website.

NB:
    This website is a static site hosted on AWS S3 and distributed via CloudFront.
    It uses HTML, CSS, and JavaScript files to deliver content to users.
